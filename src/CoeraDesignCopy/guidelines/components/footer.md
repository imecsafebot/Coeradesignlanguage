# Footer

Page footer with coera branding, tagline, social links, and legal text.

## When to Use

Use at the bottom of every web page. For kiosk/demo displays, the footer may be omitted.

## Structure (top to bottom)

1. **coera wordmark** — PP Telegraf, 32px, cream
2. **Tagline** — "Tomorrow starts with coera" — PP Mori, 16px, cream
3. **Social links** — LinkedIn, Instagram, YouTube, X — PP Mori, 14px, warm gray
4. **"Contact us" button** — orange-outlined pill button
5. **Legal text** — "2026 coera. All rights reserved." — PP Mori, 12px, warm gray at 60% opacity

## Visual Properties

| Property | Value |
|----------|-------|
| Background | Black (#000000) with subtle orange radial gradient overlay |
| Padding top | 80px |
| Padding bottom | 60px |
| Padding horizontal | 60px |
| Item spacing | 40px |
| Layout | Vertical, center-aligned |
| Width | Full viewport width |

## Rules

- Always the very last element on the page
- Tagline is always "Tomorrow starts with coera" — never change it
- Social links use warm gray, not cream — they are secondary
- Legal text at 60% opacity — it should recede
- Do not add navigation links in the footer for kiosk displays
