# Foundations — Overview

Design tokens that define the coera visual language. These are the atomic values — never use hardcoded values.

## Available Foundation Files

| File | What It Covers |
|------|---------------|
| `color.md` | Brand palette, semantic colors, usage decision trees |
| `typography.md` | PP Telegraf + PP Mori, type scale, style names |
| `spacing.md` | Spacing tokens, border radius, grid system |
| `effects.md` | Liquid glass effect, background blur |

## Key Principle

All visual values must come from tokens. Zero hardcoded hex codes, pixel values, or arbitrary numbers.
